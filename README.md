# PGO Git Exercise

This repository contains the solution for the Git exercise.  
The project demonstrates basic Git operations such as creating repositories, working with branches, merging changes, and resolving merge conflicts.

## Stages of the exercise

1. Initialize a local Git repository and create the project structure.
2. Publish the repository on GitHub.
3. Create and merge the feature/readme branch.
4. Create and merge the feature/info branch.
5. Resolve a merge conflict in README.md.
